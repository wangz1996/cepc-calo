# CEPC-CALO
Simulation Framework for CEPC calorimeters

Zhen Wang < wangz1996@sjtu.edu.cn >

## Environment
Use cvmfs
```
	source /cvmfs/sft.cern.ch/lcg/views/LCG_97rc4python3/x86_64-centos7-gcc9-opt/setup.sh
```

## Usage
Run simulation
```
	calo -c default.yaml
```

Print default config yaml
```
	calo -p
```

Print help message
```
	calo -h
```
